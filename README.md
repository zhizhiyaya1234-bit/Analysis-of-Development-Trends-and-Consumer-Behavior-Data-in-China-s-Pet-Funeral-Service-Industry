This project focuses on the pet funeral theme, completing multi-platform data scraping, organization, and visualization. 
It covers Dianping (大众点评) pet funeral store information, Weibo (微博) comments related to pet funerals, Bilibili (哔哩哔哩) danmaku from pet funeral videos, and product data from pet funeral brand official websites (dearprt.tw). 
The final outputs include dataset samples, scraping code, data cleaning code, and data visualization code.
